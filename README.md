## Getting Started

```bash
# Clone
git clone https://github.com/italia-commits/prospect.git
cd prospect

# Backend
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Database
docker compose up -d db

# Run migrations
alembic upgrade head

# Start server
uvicorn app.main:app --reload

# Frontend (Phase 2)
cd frontend
npm install
npm run dev
```
